Projects

Inflation and Corporate Profits Analysis

  • Code

The Inflation and Corporate Profits analysis project utilizes Jupyter Notebooks for data analysis and visualization, while MSSQL was utilized for data transformation. Libraries such as seaborn, plotly, matplotlib, and numpy were utilized to perform various analysis. This project aimed to observe the various relationships between inflation, price of goods, and corporate profits.

I utilized the third-party FRED API in order to search and import large scale datasets pertaining to total Coporate Profits and Consumer Price Indexes for all goods, gasoline, and Meats, Poultry, Fish and Eggs. This data was exported to MSSQL where the datasets were merged with publicly available data pertaining to the average price of gasoline per gallon, the average price of eggs per dozen, gross profits of Cal-Maine Foods, and gross profits of Exxon Mobil. The merged dataset was imported back into Jupyter Notebooks where I utilized libraries such as seaborn, matplotlib, plotly, and numby in order to perform analysis such as correlation heatmaps, time series analysis, linear regressions, and multivariate regressions. These analysis were performed in order to examine the relationships between inflation, the cost of goods, and corporate profits.

COVID-19 Global Data Analysis
  • Dashboard
  • Code

The COVID-19 data analysis project utilizes SQL for data transformation and cleaning, and Tableau for data visualization. This dashboard presents data such as total deaths and vaccinations, deaths by Continent, vaccinations by country, and a time series analysis and forecast of vaccination trends.

I gathered a large scale COVID-19 dataset and explored, cleaned, and transformed the data into structured tables using Microsoft SQL Server. Data such as aggregate deaths and vaccinations, death and vaccination percentages, location data and time series data were utilized for analysis and visualization. Intuitive dashboards and visualizations were created using Tableau, conveying key insights and trends in the COVID-19 data.

Craft Brewery Analysis in the United States
  • Dashboard
  • Code

The Craft Brewery data analysis project employs SQL for data exploration, cleaning and transformation, and uses Tableau for visualization. This dashboard presents insights such as total amount of unique beers and breweries in the United States, the top 10 states by brewery count, top 10 cities by brewery count, and what style of beer is most brewed in each state.

I gathered separate datasets of craft beers and breweries in the United States. I then utilized Microsoft SQL Server to join the datasets, and clean and transform the datasets into tables that were later used for visualization and analysis. Tableau was utilized to create visualizations and dashboards to convey key insights into the craft beer and brewery data.
